spadeCopy

An AWS Lambda function for s3 bucket replication

View project on GitHub

Purpose

Use when you have an s3 bucket that you want to replicate to another bucket as soon as a file is placed in the destination. This is useful for log file archival. I can also add image processing functions that will detect images and either resize them or convert them into the destination folder. This function is only for moving.

Language

Written in Node.JS.
